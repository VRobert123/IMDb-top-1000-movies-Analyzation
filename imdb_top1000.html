stars <- rbind(star1, star2, star3, star4)

stars_c <- count(stars, star)

stars_c <- head(arrange(stars_c, desc(n)), 10)

stars_c %>% ggplot(aes(reorder(star, -n), n)) + geom_bar(stat="identity", fill="#F3CE13") + theme(axis.text.x = element_text(angle = 90, vjust = 0.5, hjust=1)) + theme(panel.background = element_blank()) + geom_text(aes(label=n)) + xlab("Star") + ylab("Number of movies")

We build a model where output variable is IMDB_Rating and input variables are Released_Year, Meta_score, Runtime and Gross.

model <- glm(IMDB_Rating ~ Released_Year + Meta_score + Runtime + Gross, 
             data = imdb, 
             family = ("gaussian"))

Residuals vs Fitted plot shows that accurancy quite equal in the middle range and with extreme values too. In the Normal Q-Q plot we can see that the dots on the fitted line except on the top area but we have no big skew in this model. You can see on the Scale-location plot there are a minimal skew in the model because dots are randomly spread out. On the residuals vs leverage plot we cannot see a lot of outliers so probably our model won’t be misled by outliers.
